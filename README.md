# Project---HR-Data-Analytics-in-Tableau
https://public.tableau.com/app/profile/bunty.debnath/viz/HRDataAnalyticsinTableau/Dashboard1
Built a comprehensive multi-view HR analytics dashboard in Tableau Public to visualize workforce composition, diversity, attrition, and departmental performance across three core departments — Human Resources, Research &amp; Development, and Sales. The dashboard enables HR teams to identify workforce trends and make data-driven people decisions .
